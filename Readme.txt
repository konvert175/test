Для запуска приложения необходимо запустить launch.bat
Приложение запросит 
- адрес страницы (при пустом адресе будет https://simbirsoft.com)
- имя таблицы, куда необходимо выполнить сохранение результата (если такая таблица уже есть - будет удалена, если нет - создана)

Для простого теста достаточно 2 раза нажать ENTER

Страница будет сохранена в каталоге Test

батник писался с расчетом, что с GIT будет клонирован репозиторий
