1.Для запуска приложения необходимо запустить launch.bat
  Приложение запросит 
	- адрес страницы (при пустом адресе будет https://simbirsoft.com)
	- имя таблицы, куда необходимо выполнить сохранение результата (если такая таблица уже есть - будет удалена, если нет - создана)
	- пользователь БД (Windows авторизация не предусмотрена)
	- пароль пользователя

2. Для простого теста достаточно 2 раза нажать ENTER

3. Страница будет сохранена в каталоге Test

4. батник писался с расчетом, что с GIT будет клонирован репозиторий

5. Пример работы с программой:
Введите URL (по умолчанию будет подставлено https://simbirsoft.com)

	Будет осуществлен подсчет слов со страницы https://simbirsoft.com
	Введите предподчтительный способ вывода результат (пусто - на экран, иначе в таблицу, имя которой будет введено
	Table1
	Введите имя пользователя
	postgres
	Введите пароль пользователя
	1111
	Результат сохранен в таблицеTable1
